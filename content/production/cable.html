Кабель производства компании «Алмаз»

Пример условного обозначения погружного теплостойкого кабеля с фторопластовой изоляцией Пример условного обозначения погружного теплостойкого кабеля КПБП

Обособленное подразделение компании «Алмаз»- «Завод Алмаз-кабель» серийно изготавливает следующие кабели:

  • кабель теплостойкий с фторопластовой изоляцией по ТУ Р2.13.095.00.000;
  • кабель теплостойкий по ТУ Р2.13.062.00.000.

Кабели могут быть поставлены:

1. По количеству жил и сечению:

  • 3x7 мм2,
  • 3x8 мм2,
  • 3x10 мм2,
  • 3x13,3 мм2,
  • 3x14,5 мм2,
  • 3x16 мм2,
  • 3x21,15 мм2,
  • 3x25 мм2,
  • 3x35 мм2.

2. По номинальному напряжению:

  • 3300 В (испытательное напряжение 18 кВ),
  • 4000 В (испытательное напряжение 21,4 кВ),
  • 4500 В (испытательное напряжение 24 кВ).

3. По строительной длине:

  • 3x7 мм2 - до 2500 м;
  • 3x8 мм2 - до 2500 м;
  • 3x10 мм2 - до 2500 м;
  • 3x13,3 мм2 - до 2300 м;
  • 3x14,5 мм2 - до 2000 м;
  • 3x16 мм2 - до 2000 м;
  • 3x21,15 мм2 - до 2000 м;
  • 3x25 мм2 - до 1800 м;
  • 3x35 мм2 - до 1500 м.

4. По длительно допустимой температуре нагрева жил и изоляции:

  • 120°С - для КПпБП–120;
  • 125°С - для КИПпБП–125;
  • 130°С - для КИПпБП–130;
  • 140°С - для КИПпБП–140;
  • 230°С - для КИФБП–230;
  • 250°С - для КИФБП–250.

5. По коррозионной стойкости бронеленты:

  • из нержавеющей стали;
  • из материала с покрытием типа Монель;
  • из ленты АПР.

Выпускаемые кабели имеют сертификаты соответствия продукции. Каждый кабельный барабан подвергается приемосдаточным испытаниям в электротехнической лаборатории с погружением в водную ванну или испытаниям на воздухе. Заказчику направляется упаковочный ярлык, паспорт, сертификат происхождения, протокол испытаний, оформленный в соответствии с ГОСТ 51777 с результатами проведенных периодических испытаний.
