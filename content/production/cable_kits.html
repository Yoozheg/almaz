Кабельные удлинители производства компании «Алмаз»

Пример условного обозначения кабельного удлинителя из КИФБП Пример условного обозначения кабельного удлинителя из КПпБП

Обособленное подразделение компании «Алмаз»- «Завод Алмаз-кабель» серийно изготавливает следующие кабельные удлинители из кабеля:

  • кабель теплостойкий с фторопластовой изоляцией по ТУ Р2.13.095.00.000;
  • кабель теплостойкий по ТУ Р2.13.062.00.000.

Кабельные удлинители могут быть поставлены:

1. По количеству жил и сечению:

  • 3x7 мм2,
  • 3x8 мм2,
  • 3x10 мм2,
  • 3x13,3 мм2,
  • 3x14,5 мм2,
  • 3x16 мм2,
  • 3x21,15 мм2,
  • 3x25 мм2,
  • 3x35 мм2.

2. По номинальному напряжению:

  • 3300 В (испытательное напряжение 18 кВ),
  • 4000 В (испытательное напряжение 21,4 кВ),
  • 4500 В (испытательное напряжение 24 кВ).

3. По строительной длине (из учета на 5 метров выше ловильной головы ЭЦН, но по заявке заказчика может быть увеличена):

  • до 100 м.

4. По длительно допустимой температуре нагрева жил и изоляции:

  • 120°С - для КПпБП–120 (шифр 04);
  • 140°С - для КИПпБП–140 (шифр 03);
  • 230°С - для КИФБП–230 (шифр 06);
  • 250°С - для КИФБП–250 (шифр 09).

5. По коррозионной стойкости бронеленты:

  • из нержавеющей стали;
  • из материала с покрытием типа Монель;
  • из ленты АПР.

6. По производителю корпуса муфты (определяется по логотипу):

  • ООО НПФ «Битек»;
  • Компания «Алмаз».

Выпускаемые кабельные удлинители имеют сертификаты соответствия продукции. Каждый кабельный удлинитель подвергается приемосдаточным испытаниям в электротехнической лаборатории с погружением в водную ванну или испытаниям на воздухе. Заказчику направляется упаковочный ярлык, паспорт, сертификат происхождения, протокол испытаний, оформленный в соответствии с ГОСТ 51777 с результатами проведенных периодических испытаний кабеля. Упаковывается кабельный удлинитель в ящик. Количество удлинителей в одном ящике оговаривается при заказе.
